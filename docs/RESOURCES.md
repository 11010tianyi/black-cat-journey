## 原型阶段策略

1. 地形 + 植被先用 Terrain3D + ProtonScatter + CC0 贴图搭建
2. 猫模型先用占位胶囊体，验证移动手感后再购买 Vibe Poly Cats
3. 废墟场景先用简单几何体，效果满意后再买 Forest Ruins ($1.99)
4. 白猫对话先用 Dialogue Manager 文本对话，后续可加立绘

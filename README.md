# Black Cat Journey / 黑猫旅程

一款唯美、低文字压力、重氛围的 3D 第三人称探索游戏。

玩家扮演一只黑色短毛中华田园猫，拥有黄色眼睛，在广阔、荒凉又温柔的世界里独自前行。旅途中遇见不同花色的猫，短暂交互、结伴、分离。一只白色长毛猫会反复出现，以哲理、温柔和偶尔的冷笑话陪伴你。

## 技术栈

- **引擎**: Godot 4.x
- **语言**: GDScript
- **资产管线**: Blender → glTF/GLB

## 项目结构

```
├── docs/           # 设计文档
│   └── SRS.md      # 软件需求规格说明书
└── game/           # Godot 项目
    ├── assets/     # 图片、模型等资源
    ├── data/       # 配置与数据文件
    ├── materials/  # 材质
    ├── scenes/     # Godot 场景文件
    └── scripts/    # GDScript 脚本
```

## 开发

用 Godot 4.6+ 打开 `game/` 目录即可运行。
